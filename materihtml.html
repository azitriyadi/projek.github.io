HTML Home

Selamat datang di Tutorial HTML!

Ini adalah tempat yang sempurna untuk memulai perjalanan Anda dalam mempelajari HTML, bahasa markup standar untuk membuat halaman web.

Apa itu HTML?

HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk membuat halaman web dan aplikasi web. HTML memberi struktur pada konten web seperti teks, gambar, dan video. Ini adalah dasar dari setiap halaman yang Anda lihat di internet.

Kenapa Anda Harus Mempelajari HTML?

HTML adalah fondasi dari semua halaman web. Mempelajari HTML akan memberi Anda keterampilan untuk membuat dan mengatur struktur halaman web. Setelah memahami HTML, Anda akan bisa menggabungkannya dengan CSS dan JavaScript untuk membuat halaman web interaktif dan responsif.

Contoh Sederhana HTML

Berikut adalah contoh dokumen HTML dasar:


<!DOCTYPE html>
<html>
  <head>
    <title>Halaman Pertama Saya</title>
  </head>
  <body>
    <h1>Selamat datang di halaman pertama saya</h1>
    <p>Ini adalah paragraf pertama saya</p>
  </body>
</html>

Elemen HTML yang Paling Umum

  • <html>: Elemen utama dalam dokumen HTML.
  • <head>: Berisi informasi metadata tentang halaman (judul, stylesheet, dll.).
  • <body>: Berisi konten utama halaman (teks, gambar, dll.).
  • <h1> hingga <h6>: Elemen heading yang digunakan untuk memberi struktur pada konten.
  • <p>: Elemen paragraf untuk teks.
  • <a>: Elemen anchor untuk membuat tautan ke halaman lain.

Penjelasan Contoh


            <!DOCTYPE html>: ini memberitahu browser bahwa dokumen ini adalah dokumen HTML.
            <html>: Elemen ini adalah elemen akar dari sebuah halaman HTML.
            <head>: Elemen ini berisi informasi meta tentang halaman HTML.
            <title>Judul Halaman</title>: Elemen ini menentukan judul halaman.
            <body>: Elemen ini berfungsi untuk semua konten yang dapat dilihat oleh pengguna.
            <h1>Ini adalah heading</h1>: Elemen ini digunakan untuk menampilkan judul.
            <p>Ini adalah paragraf.</p>: Elemen ini digunakan untuk mendefinisikan sebuah paragraf.

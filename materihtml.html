HTML Home

Selamat datang di Tutorial HTML!

Ini adalah tempat yang sempurna untuk memulai perjalanan Anda dalam mempelajari HTML, bahasa markup standar untuk membuat halaman web yang menarik dan fungsional. HTML, atau HyperText Markup Language, adalah fondasi dari semua situs web yang Anda kunjungi setiap hari. Dalam tutorial ini, kita akan membahas berbagai elemen dan atribut HTML yang memungkinkan Anda untuk menyusun dan menampilkan konten web dengan cara yang terstruktur dan mudah dipahami.

Kenapa Anda Harus Mempelajari HTML?

HTML adalah fondasi dari semua halaman web. Mempelajari HTML akan memberi Anda keterampilan untuk membuat dan mengatur struktur halaman web. Setelah memahami HTML, Anda akan bisa menggabungkannya dengan CSS dan JavaScript untuk membuat halaman web interaktif dan responsif.

Contoh Sederhana HTML

Berikut adalah contoh dokumen HTML dasar:


<!DOCTYPE html>
<html>
  <head>
    <title>Halaman Pertama Saya</title>
  </head>
  <body>
    <h1>Selamat datang di halaman pertama saya</h1>
    <p>Ini adalah paragraf pertama saya</p>
  </body>
</html>

Apa yang Akan Anda Pelajari?

Dalam tutorial ini, Anda akan belajar berbagai topik mulai dari elemen dasar HTML hingga teknik-teknik yang lebih lanjut seperti form, tabel, dan multimedia. Berikut adalah beberapa hal yang akan Anda pelajari:

  • Dasar-dasar sintaks HTML
  • Penggunaan elemen-elemen dasar HTML
  • Membuat halaman web interaktif dengan HTML dan CSS
  • Penggunaan atribut dan formulir HTML
  • Integrasi multimedia seperti gambar, video, dan audio

HTML Introduction

APA ITU HTML?

HTML adalah bahasa markup standar yang digunakan untuk membuat halaman web, dan mendeskripsikan struktur halaman web.

Elemen HTML yang Paling Umum

  • <html>: Elemen utama dalam dokumen HTML.
  • <head>: Berisi informasi metadata tentang halaman (judul, stylesheet, dll.).
  • <body>: Berisi konten utama halaman (teks, gambar, dll.).
  • <h1> hingga <h6>: Elemen heading yang digunakan untuk memberi struktur pada konten.
  • <p>: Elemen paragraf untuk teks.
  • <a>: Elemen anchor untuk membuat tautan ke halaman lain.

Contoh Dokumen HTML Sederhana


        <!DOCTYPE html>
        <html>
        <head>
            <title>Judul Halaman</title>
        </head>
        <body>
            <h1>Ini adalah heading</h1>
            <p>Ini adalah paragraf.</p>
        </body>
        </html>

Catatan: Konten di dalam bagian <body> akan ditampilkan di browser. Sedangkan konten di dalam elemen <title> akan ditampilkan di bilah judul browser atau tab halaman.

HTML Editors

Tutorial Membuat Halaman Web Pertama Menggunakan Visual Studio Code (VSCode)

Untuk memulai belajar HTML dengan menggunakan Visual Studio Code (VSCode), berikut adalah langkah-langkah yang mudah diikuti. VSCode adalah editor kode yang canggih dengan fitur-fitur seperti auto-complete, pewarnaan sintaks, dan banyak ekstensi yang memudahkan kamu dalam menulis kode HTML.

Langkah 1: Instal Visual Studio Code

Unduh Visual Studio Code (VSCode)

  • Buka situs resmi VSCode.
  • Pilih versi untuk sistem operasi yang kamu gunakan (Windows, macOS, atau Linux).
  • Ikuti petunjuk instalasi sesuai dengan sistem operasi yang kamu pilih.

Instal VSCode

  • Setelah mengunduh, jalankan file instalasi dan ikuti langkah-langkah untuk menginstal Visual Studio Code di komputermu.

Langkah 2: Membuka VSCode

  • Setelah selesai menginstal, buka aplikasi Visual Studio Code dari menu Start (Windows) atau melalui Finder (Mac).
  • Jika ini pertama kali kamu membuka VSCode, kamu akan melihat tampilan editor kosong siap digunakan untuk menulis kode.

Langkah 3: Membuat File HTML Baru

  • Buat Folder untuk Proyekmu
  • Buat sebuah folder di komputer untuk menyimpan proyek HTML kamu, misalnya "MyFirstWebsite".
  • Buat File HTML Baru
  • Klik File > New File atau tekan Ctrl + N (Windows) atau Cmd + N (Mac).
  • Setelah itu, pilih File > Save As, dan simpan dengan nama index.html di folder yang baru kamu buat.

Langkah 4: Menulis Kode HTML

Menulis Kode HTML Dasar

Di dalam file index.html, salin dan tempel kode berikut:

        
        <!DOCTYPE html>
        <html lang="id">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Halaman Pertama Saya</title>
        </head>
        <body>
            <h1>Selamat Datang di Halaman Pertama Saya!</h1>
            <p>Ini adalah paragraf pertama saya di HTML. Saya belajar membuat halaman web menggunakan Visual Studio Code.</p>
        </body>
        </html>

Penjelasan Kode:

  • <!DOCTYPE html>: Memberitahu browser bahwa dokumen ini adalah dokumen HTML.
  • <html lang="id">: Menandakan bahwa halaman ini menggunakan bahasa Indonesia.
  • <head>: Berisi informasi metadata tentang halaman web, seperti judul halaman.
  • <body>: Tempat semua konten halaman web seperti teks, gambar, dan lainnya ditampilkan.

Langkah 5: Menyimpan dan Melihat Halaman Web

  • Simpan File HTML
  • Pastikan kamu sudah menyimpan file dengan memilih File > Save atau tekan Ctrl + S (Windows) atau Cmd + S (Mac).
  • Buka Halaman di Browser
  • Buka folder tempat kamu menyimpan file index.html.
  • Klik dua kali pada file index.html. Ini akan membuka file tersebut di browser default kamu.
  • Kamu akan melihat tampilan seperti berikut:
    • Judul halaman akan terlihat di tab browser.
    • Teks "Selamat Datang di Halaman Pertama Saya!" akan ditampilkan sebagai judul besar (Heading).
    • Paragraf pertama akan muncul di bawahnya.

Langkah 6: Mengedit dan Melihat Perubahan

  • Mengedit Kode di VSCode
  • Setiap kali kamu membuat perubahan pada kode HTML di VSCode, cukup tekan Ctrl + S (Windows) atau Cmd + S (Mac) untuk menyimpannya.
  • Lihat Hasil Perubahan
  • Setelah menyimpan file, kembali ke browser dan tekan F5 atau Refresh untuk melihat perubahan yang baru saja kamu buat.

Langkah 7: Menggunakan Fitur Tambahan di VSCode

  • Auto-completion: VSCode akan memberikan saran otomatis untuk tag HTML yang ingin kamu tulis, sehingga kamu tidak perlu mengetiknya satu per satu.
  • Highlighting: Kode yang kamu tulis akan diberi warna yang berbeda untuk elemen-elemen HTML, sehingga lebih mudah dibaca dan dipahami.
  • Live Server: Jika kamu ingin melihat hasil HTML secara langsung tanpa harus membuka browser manual, kamu bisa menginstal ekstensi Live Server di VSCode untuk melihat perubahan secara langsung saat kamu mengetik.

Langkah 8: Melanjutkan Belajar HTML

Setelah kamu berhasil membuat halaman pertama menggunakan VSCode, kamu bisa melanjutkan untuk belajar elemen-elemen HTML lainnya seperti:

  • Formulir: Membuat formulir interaktif.
  • Tabel: Menyusun data dalam bentuk tabel.
  • Gambar dan Video: Menyisipkan media seperti gambar dan video ke dalam halaman web.
  • CSS: Menambahkan styling dan desain menggunakan CSS.

Kesimpulan

Dengan menggunakan Visual Studio Code (VSCode), kamu bisa belajar HTML secara efisien dan efektif. Editor ini memberikan banyak fitur canggih untuk membantu kamu menulis kode dengan lebih cepat dan tepat. Setelah membuat halaman pertama, kamu dapat terus mengembangkan kemampuanmu dengan menambahkan lebih banyak elemen HTML dan menghubungkannya dengan CSS atau JavaScript.

Selamat belajar HTML, dan semoga tutorial ini membantu kamu memulai perjalananmu dalam dunia pengembangan web!

HTML Dasar

1. Dokumen HTML

Setiap dokumen HTML dimulai dengan deklarasi jenis dokumen untuk memberi tahu browser bahwa dokumen yang akan diproses adalah HTML5. Deklarasi ini ditulis sebagai <!DOCTYPE html> dan harus berada di bagian paling atas dokumen.

Setelah deklarasi jenis dokumen, dokumen HTML dimulai dengan tag pembuka <html> dan diakhiri dengan tag penutup </html>. Semua elemen dalam halaman HTML diletakkan di dalam tag <html>. Sedangkan bagian yang akan ditampilkan di browser, seperti teks, gambar, dan elemen-elemen lain, berada di dalam tag <body>.

Contoh Dasar HTML
<!DOCTYPE html>
    <html>
    <body>
    
    <h1>Judul Pertama Saya</h1>
    <p>Paragraf pertama saya.</p>
    
    </body>
    </html>

Penjelasan Kode:

  • <!DOCTYPE html>: Mendeklarasikan bahwa dokumen ini adalah dokumen HTML5 yang digunakan pada browser modern.
  • <html> dan </html>: Menandakan awal dan akhir dari dokumen HTML, menyatakan bahwa segala elemen yang ada di dalamnya adalah bagian dari dokumen HTML.
  • <body> dan </body>: Bagian dari dokumen yang akan ditampilkan oleh browser. Semua konten yang ingin ditampilkan, seperti teks, gambar, atau elemen lain, harus berada dalam tag <body>.
  • <h1>: Tag ini digunakan untuk menandakan judul tingkat pertama atau heading terbesar. Judul ini akan muncul dengan ukuran font terbesar.
  • <p>: Digunakan untuk mendefinisikan paragraf. Semua teks yang ingin dipisah menjadi blok paragraf harus berada di dalam tag <p>.
2. Deklarasi <!DOCTYPE>

Deklarasi <!DOCTYPE> adalah instruksi penting untuk memberitahu browser jenis dokumen yang digunakan. Dengan menulis <!DOCTYPE html>, kita memberitahukan browser bahwa dokumen ini menggunakan HTML5. Meskipun hal ini tidak memengaruhi tampilan halaman, deklarasi ini sangat penting untuk memastikan bahwa dokumen ditafsirkan dengan benar.

<!DOCTYPE html>
3. Judul dalam HTML (Headings)

Judul dalam HTML ditandai dengan tag <h1> hingga <h6>. Tag <h1> digunakan untuk mendefinisikan judul utama yang paling penting, dan tag <h6> digunakan untuk judul dengan tingkat prioritas yang paling rendah. Judul tingkat lebih tinggi seperti <h1> memberi penekanan lebih pada konten yang ada di dalamnya.

Judul adalah cara yang baik untuk menyusun informasi dalam halaman web dan memberi struktur yang jelas. Hal ini juga penting untuk SEO (Search Engine Optimization), karena judul membantu mesin pencari memahami struktur dan konteks halaman.

Contoh Judul
<h1>Ini adalah Judul 1</h1>
    <h2>Ini adalah Judul 2</h2>
    <h3>Ini adalah Judul 3</h3>
    <h4>Ini adalah Judul 4</h4>
    <h5>Ini adalah Judul 5</h5>
    <h6>Ini adalah Judul 6</h6>
4. Paragraf dalam HTML

Tag <p> digunakan untuk mendefinisikan paragraf dalam HTML. Setiap blok teks yang ingin dipisahkan dan diatur dalam satu bagian yang terstruktur harus ditulis di dalam tag <p>.

Paragraf sering digunakan untuk menyusun informasi yang lebih panjang, memberi ruang dan jarak antar konten di halaman web.

Contoh Paragraf
<p>Ini adalah paragraf pertama.</p>
    <p>Ini adalah paragraf kedua.</p>
5. Tautan (Links) dalam HTML

Tautan dalam HTML dibuat dengan menggunakan tag <a>. Tautan digunakan untuk menghubungkan halaman web dengan halaman lainnya, baik itu halaman di situs yang sama ataupun ke situs yang berbeda. Atribut href dalam tag <a> digunakan untuk menentukan tujuan tautan tersebut.

Contoh Tautan
<a href="https://www.w3schools.com">Ini adalah sebuah tautan</a>
6. Gambar dalam HTML

Untuk menyisipkan gambar dalam halaman HTML, kita menggunakan tag <img>. Tag ini adalah elemen kosong, yang berarti tidak memiliki tag penutup. Atribut yang digunakan dalam tag <img> antara lain src untuk menunjukkan sumber gambar, alt untuk menyediakan teks alternatif jika gambar gagal dimuat, serta width dan height untuk mengatur ukuran gambar.

Contoh Gambar
<img src="w3schools.jpg" alt="W3Schools.com" width="104" height="142">
7. Melihat Sumber HTML

Kamu dapat memeriksa dan mempelajari kode sumber HTML dari halaman web yang ada dengan beberapa cara. Salah satu cara yang paling umum adalah dengan menekan CTRL + U pada halaman di browser, atau mengklik kanan pada halaman dan memilih "View Page Source".

Selain itu, banyak browser juga menyediakan fitur "Inspect" yang memungkinkanmu melihat elemen HTML yang digunakan dalam halaman, serta mengeditnya secara langsung untuk eksperimen dan pembelajaran.

Kesimpulan
  • HTML adalah bahasa markup dasar untuk membuat halaman web.
  • Setiap dokumen HTML diawali dengan deklarasi <!DOCTYPE html>, diikuti oleh tag <html>, dan konten halaman berada di dalam tag <body>.
  • Melalui fitur "View Source" dan "Inspect", kita dapat melihat dan memahami bagaimana struktur HTML dan elemen-elemen lainnya di halaman web dibangun.

Elemen HTML

Elemen HTML adalah unit dasar dari struktur sebuah halaman web yang terdiri dari tag awal, konten, dan tag akhir. Elemen ini digunakan untuk mendefinisikan berbagai jenis bagian dalam halaman web, seperti teks, gambar, tautan, dan elemen interaktif lainnya. Tag awal menunjukkan jenis elemen yang dimulai, konten berisi data yang ingin ditampilkan, dan tag akhir menandakan bahwa elemen tersebut telah selesai. Dengan menggunakan elemen-elemen HTML ini, pengembang dapat membangun struktur dan tata letak halaman web yang sesuai dengan kebutuhan pengguna.

Struktur Elemen HTML

Elemen HTML adalah segalanya mulai dari tag awal hingga tag akhir, yang terlihat seperti ini:

        <tagname>Konten berada di sini...</tagname>
<div>Ini adalah sebuah div</div>

Contoh Elemen HTML


        <h1>Judul Pertama Saya</h1>
        <p>Paragraf pertama saya.</p>
<p>Paragraf kedua saya dengan beberapa teks tambahan.</p>

Penjelasan:

  • Tag Awal: <h1> dan <p> adalah tag awal yang menandakan dimulainya elemen tersebut.
  • Konten Elemen: "Judul Pertama Saya" dan "Paragraf pertama saya" adalah konten yang berada di dalam tag.
  • Tag Akhir: </h1> dan </p> adalah tag akhir yang menandakan berakhirnya elemen tersebut.

Tabel Penjelasan:

Tag Awal Konten Elemen Tag Akhir
<h1> Judul Pertama Saya </h1>
<p> Paragraf pertama saya </p>

Elemen Kosong (Empty Elements)

Beberapa elemen HTML tidak memiliki konten, dan ini disebut elemen kosong (empty elements). Elemen kosong tidak memerlukan tag penutup. Contoh elemen kosong adalah tag <br> yang digunakan untuk membuat pemisah baris baru dalam teks.

        <p>Ini adalah paragraf dengan <br> pemisah baris.</p>
<br>Teks setelah pemisah baris.

Elemen HTML yang Ter-nest (Bersarang)

Elemen HTML dapat bersarang, artinya sebuah elemen bisa berada di dalam elemen lainnya. Semua dokumen HTML terdiri dari elemen-elemen yang bersarang.

Contoh Dokumen HTML Sederhana

        <!DOCTYPE html>
        <html>
          <body>
            <h1>Judul Pertama Saya</h1>
            <p>Paragraf pertama saya.</p>
          </body>
        </html>
<html> <head><title>Halaman Judul</title></head> <body> <p>Selamat datang di halaman saya</p> </body> </html>

Atribut HTML

Atribut HTML memberikan informasi tambahan tentang elemen HTML.

Atribut href

Tag <a> mendefinisikan hyperlink. Atribut href menentukan URL halaman yang akan dituju oleh link.

 <a href="https://www.w3schools.com">Kunjungi W3Schools</a>

Atribut href digunakan untuk mendefinisikan alamat URL tujuan ketika pengguna mengklik sebuah hyperlink. Atribut ini umumnya digunakan dalam tag <a>.

Atribut src

Tag <img> digunakan untuk menyisipkan gambar dalam halaman HTML. Atribut src menentukan jalur gambar yang akan ditampilkan.

<img src="img_girl.jpg">

Ada dua cara untuk menentukan URL dalam atribut src:

  • URL Absolut: Menautkan gambar eksternal yang dihosting di situs lain. Contoh: src="https://www.w3schools.com/images/img_girl.jpg".
  • URL Relatif: Menautkan gambar yang dihosting di dalam situs web. Contoh: src="img_girl.jpg" atau src="/images/img_girl.jpg".

Atribut width dan height

Tag <img> sebaiknya juga mengandung atribut width dan height, yang menentukan lebar dan tinggi gambar (dalam piksel).

 <img src="img_girl.jpg" width="500" height="600">

Atribut width dan height mengatur ukuran gambar yang ditampilkan. Anda dapat menyesuaikan keduanya agar sesuai dengan desain halaman web Anda.

Atribut alt

Atribut alt yang wajib ada untuk tag <img> menentukan teks alternatif untuk gambar, jika gambar tidak dapat ditampilkan.

 <img src="img_girl.jpg" alt="Gadis dengan jaket">

Atribut alt membantu pengguna dengan gangguan penglihatan, atau ketika gambar tidak dapat dimuat, karena akan ditampilkan teks alternatif yang menggambarkan gambar tersebut.

Atribut style

Atribut style digunakan untuk menambahkan gaya pada elemen, seperti warna, font, ukuran, dan lainnya.

<p style="color:red;">Ini adalah paragraf merah.</p>

Atribut style memungkinkan Anda untuk menambahkan CSS langsung ke elemen HTML. Anda dapat mengubah berbagai gaya visual seperti warna, margin, font, dll.

Atribut lang

Anda sebaiknya selalu menyertakan atribut lang di dalam tag <html>, untuk mendeklarasikan bahasa halaman Web.

<html lang="id">

Atribut lang memberi tahu browser dan mesin pencari bahasa yang digunakan di halaman tersebut. Ini berguna untuk meningkatkan aksesibilitas dan SEO halaman.

Atribut title

Atribut title mendefinisikan informasi tambahan tentang elemen. Nilai atribut title akan ditampilkan sebagai tooltip ketika Anda mengarahkan kursor ke elemen.

<p title="Ini adalah tooltip">Ini adalah paragraf.</p>

Atribut title sering digunakan untuk memberikan keterangan lebih lanjut mengenai elemen, yang dapat dilihat pengguna ketika mereka mengarahkan kursor ke elemen tersebut.

Saran: Gunakan Atribut dengan Huruf Kecil

Standar HTML tidak mewajibkan penggunaan huruf kecil untuk nama atribut, tetapi W3C merekomendasikan penggunaan huruf kecil dalam HTML dan mengharuskannya dalam tipe dokumen yang lebih ketat seperti XHTML.

Saran: Selalu Gunakan Tanda Kutip untuk Nilai Atribut

W3C merekomendasikan penggunaan tanda kutip dalam HTML, dan mengharuskannya dalam tipe dokumen yang lebih ketat seperti XHTML.

<a href="https://www.w3schools.com/html/">Kunjungi tutorial HTML kami</a>

Ini adalah cara yang benar untuk menulis atribut dengan nilai yang mengandung spasi atau karakter khusus.

Tanda Kutip Tunggal atau Ganda?

Tanda kutip ganda di sekitar nilai atribut adalah yang paling umum dalam HTML, tetapi tanda kutip tunggal juga bisa digunakan.

<p title='John "ShotGun" Nelson'>

Anda bisa memilih untuk menggunakan tanda kutip tunggal atau ganda sesuai dengan kebutuhan. Jika nilai atribut mengandung tanda kutip ganda, lebih baik menggunakan tanda kutip tunggal.

Ringkasan Bab

  • Semua elemen HTML dapat memiliki atribut.
  • Atribut href pada <a> menentukan URL halaman yang dituju oleh link.
  • Atribut src pada <img> menentukan jalur gambar yang akan ditampilkan.
  • Atribut width dan height pada <img> memberikan informasi ukuran gambar.
  • Atribut alt pada <img> menyediakan teks alternatif untuk gambar.
  • Atribut style digunakan untuk menambahkan gaya pada elemen, seperti warna, font, ukuran, dan lainnya.
  • Atribut lang pada tag <html> mendeklarasikan bahasa halaman Web.
  • Atribut title mendefinisikan informasi tambahan tentang elemen.

Judul HTML

Judul HTML adalah judul atau subjudul yang ingin Anda tampilkan di halaman web. Penggunaan judul yang tepat sangat penting dalam memberikan struktur pada dokumen web Anda dan membantu mesin pencari memahami konten halaman.

Contoh Judul:

  • Heading 1
  • Heading 2
  • Heading 3
  • Heading 4
  • Heading 5
  • Heading 6

Judul ini memberikan hirarki informasi pada halaman Anda. Sebagai contoh, <h1> biasanya digunakan untuk judul utama halaman, sementara <h6> lebih sering digunakan untuk judul bagian kecil atau subjudul yang kurang penting.

Tag Judul HTML

Judul HTML didefinisikan dengan tag <h1> hingga <h6>. Semakin kecil nomor tag, semakin kurang penting judul tersebut dalam hierarki konten halaman.

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

Berikut adalah contoh penggunaan tag <h1> hingga <h6> di dalam HTML. Anda sebaiknya mengikuti urutan dari <h1> hingga <h6> untuk memberikan struktur yang jelas pada halaman web.

Catatan: Browser secara otomatis menambahkan beberapa ruang putih (margin) sebelum dan sesudah sebuah judul, untuk memberi tampilan yang lebih rapi dan terpisah antara konten.

Pentingnya Judul

Judul-judul ini tidak hanya membantu pengguna untuk memahami konten halaman dengan cepat, tetapi juga penting bagi mesin pencari. Mesin pencari seperti Google menggunakan judul untuk mengindeks halaman dan menentukan relevansi halaman dengan kata kunci pencarian.

Selain itu, judul-judul ini sangat membantu pengguna yang memindai halaman. Sebagai contoh, pengguna yang sedang mencari informasi tertentu akan lebih mudah menemukan bagian yang relevan jika judul-judul digunakan dengan tepat.

 <h1>Judul Utama</h1>
<h2>Subjudul Sekunder</h2>
<h3>Subbagian Terkait</h3>

Gunakan <h1> untuk judul utama halaman, diikuti dengan <h2> untuk subjudul, dan <h3> hingga <h6> untuk subbagian yang lebih kecil. Menjaga hirarki ini akan sangat membantu dalam memudahkan navigasi pengguna dan meningkatkan SEO halaman Anda.

Catatan: Jangan gunakan tag heading hanya untuk membuat teks lebih besar atau tebal. Tujuan utama tag <h1> hingga <h6> adalah untuk struktur dokumen, bukan untuk dekorasi teks.

Menentukan Ukuran Judul

Setiap judul HTML memiliki ukuran default sesuai dengan tingkatannya. Namun, jika Anda ingin mengubah ukuran judul secara manual, Anda dapat menggunakan atribut style dengan properti CSS font-size.

<h1 style="font-size:60px;">Heading 1</h1>

Di sini, ukuran font judul <h1> telah diubah menjadi 60 piksel. Anda bisa mengubah ukuran judul dengan cara yang sama, menyesuaikan dengan desain halaman yang Anda inginkan.

Komentar HTML

Komentar HTML tidak akan ditampilkan di browser, tetapi dapat membantu mendokumentasikan kode sumber HTML Anda.

Tag Komentar HTML

Anda dapat menambahkan komentar ke dalam kode HTML menggunakan sintaks berikut:

 <!-- Ini adalah komentar -->

Perhatikan bahwa terdapat tanda seru (!) di awal tag komentar, namun tidak ada di tag penutupnya. Komentar ini tidak akan muncul di tampilan browser.

Menambahkan Komentar

Dengan komentar, Anda bisa menambahkan notifikasi dan pengingat dalam kode HTML Anda. Komentar tidak akan mempengaruhi tampilan halaman atau fungsionalitas halaman web, namun sangat berguna untuk memberi penjelasan atau catatan dalam kode HTML Anda.

  • <!-- Ini adalah komentar -->
  • <!-- Ingat untuk memperbarui konten halaman ini -->

Menyembunyikan Konten Menggunakan Komentar

Dengan menggunakan komentar, Anda dapat menyembunyikan bagian konten yang tidak ingin ditampilkan sementara waktu. Ini berguna saat Anda ingin menonaktifkan bagian tertentu dalam HTML, misalnya saat debugging atau percakapan proyek.

Contoh menyembunyikan sebuah elemen:

<!-- <p>Ini adalah paragraf yang disembunyikan</p> -->

Semua yang ada di antara <!-- dan --> akan disembunyikan dari browser. Ini memungkinkan Anda untuk menonaktifkan elemen sementara tanpa menghapusnya dari kode.

Menonaktifkan Kode Sementara

Jika Anda sedang mengerjakan proyek atau mencoba berbagai solusi, Anda bisa menonaktifkan kode HTML sementara dengan cara mengomentari kode tersebut. Ini sangat berguna saat debugging atau mencoba berbagai opsi tanpa menghapus kode yang ada.

<!-- <p>Konten ini sementara dinonaktifkan</p> -->

Menambahkan Penjelasan Menggunakan Komentar

Komentar sangat berguna untuk memberi penjelasan atau catatan kepada pengembang lain atau diri Anda di masa depan. Anda bisa menulis penjelasan yang menjelaskan kode Anda tanpa menambah beban tampilan di browser. Ini juga sangat membantu dalam kolaborasi tim.

<!-- Penjelasan: Menambahkan gambar di bawah ini -->
<img src="gambar.jpg" alt="Deskripsi gambar">

WARNA HTML

Warna HTML dapat ditentukan dengan nama warna yang sudah ada, atau menggunakan nilai RGB, HEX, HSL, RGBA, atau HSLA.

Nama Warna

  • Tomato
  • Orange
  • DodgerBlue
  • MediumSeaGreen
  • Gray
  • SlateBlue
  • Violet
  • LightGray

HTML mendukung 140 nama warna standar.

Warna dengan Nilai RGB

Warna dapat ditentukan menggunakan model RGB (Red, Green, Blue), dengan nilai dari 0 hingga 255 untuk setiap warna. Formatnya adalah rgb(red, green, blue).

rgb(255, 99, 71)

Contoh:

  • rgb(255, 99, 71) - Tomato
  • rgb(255, 165, 0) - Orange
  • rgb(30, 144, 255) - DodgerBlue
  • rgb(60, 179, 113) - MediumSeaGreen

Warna dengan Nilai HEX

Nilai HEX adalah cara lain untuk mendefinisikan warna. Formatnya adalah #RRGGBB, di mana RR, GG, dan BB adalah pasangan dua digit hexadecimal yang mewakili warna merah, hijau, dan biru.

#FF6347

Contoh:

  • #FF6347 - Tomato
  • #FFA500 - Orange
  • #1E90FF - DodgerBlue
  • #3CB371 - MediumSeaGreen

Warna dengan Nilai HSL

Warna juga dapat ditentukan menggunakan model HSL (Hue, Saturation, Lightness). Formatnya adalah hsl(hue, saturation%, lightness%).

hsl(9, 100%, 64%)

Contoh:

  • hsl(9, 100%, 64%) - Tomato
  • hsl(39, 100%, 50%) - Orange
  • hsl(210, 100%, 56%) - DodgerBlue
  • hsl(120, 60%, 50%) - MediumSeaGreen

Warna dengan Nilai RGBA

RGBA adalah varian dari model RGB yang juga memungkinkan Anda untuk mengatur tingkat transparansi (alpha). Formatnya adalah rgba(red, green, blue, alpha), di mana alpha adalah nilai antara 0 (sepenuhnya transparan) dan 1 (sepenuhnya opak).

rgba(255, 99, 71, 0.5)

Contoh:

  • rgba(255, 99, 71, 0.5) - Tomato dengan transparansi
  • rgba(255, 165, 0, 0.7) - Orange dengan transparansi
  • rgba(30, 144, 255, 0.6) - DodgerBlue dengan transparansi
  • rgba(60, 179, 113, 0.8) - MediumSeaGreen dengan transparansi

Warna dengan Nilai HSLA

HSLA adalah varian dari model HSL yang juga memungkinkan Anda mengatur transparansi. Formatnya adalah hsla(hue, saturation%, lightness%, alpha).

hsla(9, 100%, 64%, 0.5)

Contoh:

  • hsla(9, 100%, 64%, 0.5) - Tomato dengan transparansi
  • hsla(39, 100%, 50%, 0.7) - Orange dengan transparansi
  • hsla(210, 100%, 56%, 0.6) - DodgerBlue dengan transparansi
  • hsla(120, 60%, 50%, 0.8) - MediumSeaGreen dengan transparansi

Contoh Penggunaan Warna dalam HTML

Berikut adalah contoh penggunaan warna dalam elemen HTML menggunakan CSS:


<style>
    body {
        background-color: rgb(255, 99, 71); /* Warna Tomato */
    }
    h1 {
        color: #FF6347; /* Warna Tomato dalam HEX */
    }
    p {
        color: hsl(39, 100%, 50%); /* Warna Orange */
    }
</style>

Apa itu CSS?

Cascading Style Sheets (CSS) digunakan untuk memformat layout sebuah halaman web. Dengan CSS, Anda dapat mengontrol berbagai aspek seperti warna, font, ukuran teks, jarak antar elemen, posisi elemen, gambar latar belakang, dan banyak lagi.

Tip: Kata "cascading" berarti bahwa gaya yang diterapkan pada elemen induk juga akan diterapkan ke semua elemen anak di dalam induk tersebut.

Menggunakan CSS

CSS dapat ditambahkan ke dokumen HTML dengan 3 cara:

  • Inline - dengan menggunakan atribut style di dalam elemen HTML.
  • Internal - dengan menggunakan elemen <style> di dalam bagian <head>.
  • External - dengan menggunakan elemen <link> untuk menghubungkan ke file CSS eksternal.

Inline CSS

CSS inline digunakan untuk menerapkan gaya unik pada satu elemen HTML. Inline CSS menggunakan atribut style dari elemen HTML.

<h1 style="color:blue;">A Heading Berwarna Biru</h1>
<p style="color:red;">Sebuah paragraf merah.</p>

Internal CSS

Tautan HTML

Tautan ditemukan di hampir semua halaman web. Tautan memungkinkan pengguna untuk mengklik dan berpindah dari satu halaman ke halaman lainnya.

Tautan HTML - Hyperlink

Tautan HTML adalah hyperlink. Anda dapat mengklik tautan dan berpindah ke dokumen lain. Saat Anda menggerakkan mouse di atas tautan, kursor mouse akan berubah menjadi tangan kecil. Catatan: Tautan tidak harus berupa teks. Tautan bisa berupa gambar atau elemen HTML lainnya!

Sintaks Tautan HTML

Tag HTML <a> mendefinisikan sebuah hyperlink. Berikut adalah sintaksnya:

<a href="url">teks tautan</a>

Atribut yang paling penting dari elemen adalah atribut href, yang menunjukkan tujuan tautan. Teks tautan adalah bagian yang akan terlihat oleh pembaca. Saat mengklik teks tautan, pembaca akan diarahkan ke alamat URL yang ditentukan.

Contoh

Contoh ini menunjukkan cara membuat tautan ke W3Schools.com:

<a href="https://www.w3schools.com/">Kunjungi W3Schools.com!</a>

Tautan HTML - Atribut target

Secara default, halaman yang ditautkan akan ditampilkan di jendela browser yang sama. Untuk mengubahnya, Anda harus menentukan target lain untuk tautan.

Atribut target menentukan di mana dokumen yang ditautkan akan dibuka.

Atribut target dapat memiliki salah satu nilai berikut:

  • _self - Default. Membuka dokumen di jendela/tab yang sama dengan tempat tautan diklik
  • _blank - Membuka dokumen di jendela atau tab baru
  • _parent - Membuka dokumen di frame induk
  • _top - Membuka dokumen di seluruh jendela browser

Contoh

Gunakan target="_blank" untuk membuka dokumen yang ditautkan di jendela atau tab baru:

<a href="https://www.w3schools.com/" target="_blank">Kunjungi W3Schools!</a>

Images HTML

Gambar dapat memperbaiki desain dan tampilan halaman web.

Contoh

<img src="pic_trulli.jpg" alt="Trulli Italia">
<img src="img_girl.jpg" alt="Gadis dengan jaket">
<img src="img_chania.jpg" alt="Bunga di Chania">

Sintaks HTML Gambar

Tag HTML <img> digunakan untuk menyematkan gambar dalam halaman web.

Gambar tidak benar-benar disisipkan ke dalam halaman web; gambar dihubungkan ke halaman web. Tag <img> membuat ruang penampung untuk gambar yang dirujuk.

Tag <img> adalah tag kosong, hanya berisi atribut, dan tidak memiliki tag penutup.

Tag <img> memiliki dua atribut yang wajib ada:

  • src - Menentukan path menuju gambar
  • alt - Menentukan teks alternatif untuk gambar

Sintaks

<img src="url" alt="teksalternatif">

Atribut src

Atribut src yang wajib digunakan, menentukan path (URL) menuju gambar.

Catatan: Saat halaman web dimuat, browser akan mengambil gambar dari server web dan menyisipkannya ke dalam halaman. Oleh karena itu, pastikan gambar tetap berada di tempat yang benar agar pengunjung tidak mendapatkan ikon link rusak. Ikon link rusak dan teks alt akan ditampilkan jika browser tidak dapat menemukan gambar.

<img src="img_chania.jpg" alt="Bunga di Chania">

Atribut alt

Atribut alt yang wajib digunakan menyediakan teks alternatif untuk gambar, jika pengguna tidak dapat melihat gambar tersebut (karena koneksi yang lambat, kesalahan pada atribut src, atau jika pengguna menggunakan pembaca layar).

Nilai dari atribut alt sebaiknya menjelaskan gambar tersebut:

<img src="img_chania.jpg" alt="Bunga di Chania">

Jika browser tidak dapat menemukan gambar, maka akan menampilkan nilai dari atribut alt:

<img src="wrongname.gif" alt="Bunga di Chania">

Tip: Pembaca layar adalah program perangkat lunak yang membaca kode HTML dan memungkinkan pengguna "mendengarkan" konten. Pembaca layar berguna bagi orang yang memiliki keterbatasan penglihatan atau kesulitan belajar.

Ukuran Gambar - Lebar dan Tinggi

Anda dapat menggunakan atribut style untuk menentukan lebar dan tinggi gambar.

<img src="img_girl.jpg" alt="Gadis dengan jaket" style="width:500px;height:600px;">

Alternatifnya, Anda dapat menggunakan atribut width dan height:

<img src="img_girl.jpg" alt="Gadis dengan jaket" width="500" height="600">

Atribut width dan height selalu menentukan lebar dan tinggi gambar dalam piksel.

Catatan: Selalu tentukan lebar dan tinggi gambar. Jika lebar dan tinggi tidak ditentukan, halaman web mungkin berkedip saat gambar dimuat.

Lebar dan Tinggi, atau Style?

Atribut width, height, dan style semuanya valid dalam HTML.

Namun, kami menyarankan untuk menggunakan atribut style. Ini mencegah stylesheet mengubah ukuran gambar:


        <!DOCTYPE html>
        <html>
        <head>
        <style>
        img { width: 100%; }
        </style>
        </head>
        <body>

        <img src="html5.gif" alt="Ikon HTML5" width="128" height="128">

        <img src="html5.gif" alt="Ikon HTML5" style="width:128px;height:128px;">

        </body>
        </html>

Gambar dalam Folder Lain

Jika gambar Anda berada dalam sub-folder, Anda harus menyertakan nama folder dalam atribut src:

<img src="/images/html5.gif" alt="Ikon HTML5" style="width:128px;height:128px;">

Gambar di Server/Situs Lain

Beberapa situs web mengarah ke gambar di server lain.

Untuk mengarahkan ke gambar di server lain, Anda harus menentukan URL absolut (URL lengkap) dalam atribut src:

<img src="https://www.w3schools.com/images/w3schools_green.jpg" alt="W3Schools.com">

Catatan tentang gambar eksternal: Gambar eksternal mungkin memiliki hak cipta. Jika Anda tidak mendapatkan izin untuk menggunakannya, Anda bisa melanggar hukum hak cipta. Selain itu, Anda tidak dapat mengontrol gambar eksternal tersebut; gambar tersebut bisa saja tiba-tiba dihapus atau diubah.

Gambar Animasi

HTML mendukung GIF animasi:

<img src="programming.gif" alt="Manusia Komputer" style="width:48px;height:48px;">

Gambar sebagai Link

Favicon HTML

Favicon adalah gambar kecil yang ditampilkan di sebelah judul halaman di tab browser.

Cara Menambahkan Favicon di HTML

Anda dapat menggunakan gambar apa saja sebagai favicon. Anda juga dapat membuat favicon Anda sendiri di situs seperti https://www.favicon.cc.

Tip: Favicon adalah gambar kecil, jadi sebaiknya pilih gambar yang sederhana dengan kontras tinggi.

Format Gambar untuk Favicon

Favicon biasanya digunakan dalam format gambar .ico atau .png. Meskipun Anda bisa menggunakan format lain, .ico adalah format yang paling umum dan kompatibel di semua browser.

  • Format paling umum: .ico
  • Format lain: .png, .jpg, .gif

Menambahkan Favicon dengan Tag HTML

Untuk menambahkan favicon ke halaman HTML, gunakan tag <link> di dalam <head> bagian dari dokumen HTML Anda. Berikut adalah contoh menambahkan favicon:


<link rel="icon" href="favicon.ico" type="image/x-icon">

Tag <link> dengan atribut rel="icon" memberi tahu browser bahwa gambar tersebut adalah favicon untuk halaman web tersebut. Atribut href menentukan lokasi gambar favicon, sedangkan atribut type menunjukkan tipe gambar.

Menambahkan Favicon dengan Format PNG

Jika Anda menggunakan format PNG untuk favicon, Anda dapat menggunakan kode berikut:


<link rel="icon" href="favicon.png" type="image/png">

Pastikan gambar favicon memiliki ukuran yang sesuai. Ukuran standar favicon adalah 16x16 piksel, tetapi Anda juga bisa menggunakan 32x32 piksel atau 48x48 piksel untuk kualitas yang lebih baik di layar dengan resolusi tinggi (seperti retina display).

Menambahkan Favicon untuk Apple Devices

Jika Anda ingin favicon ditampilkan dengan baik di perangkat Apple, Anda dapat menambahkan ikon khusus dengan menambahkan tag <link> untuk favicon Apple:


<link rel="apple-touch-icon" href="apple-icon.png">

Dengan menambahkan tag ini, ikon akan muncul di layar home perangkat Apple ketika pengguna menambahkan situs Anda ke layar utama mereka.

Menambahkan Favicon untuk Android Devices

Untuk perangkat Android, Anda bisa menambahkan favicon dalam format PNG yang lebih besar dengan menambahkan tag berikut:


<link rel="icon" sizes="192x192" href="android-icon.png">

Contoh Lengkap Menambahkan Favicon

Berikut adalah contoh lengkap menambahkan berbagai favicon untuk berbagai perangkat:


<head>
    <link rel="icon" href="favicon.ico" type="image/x-icon">
    <link rel="icon" href="favicon.png" type="image/png">
    <link rel="apple-touch-icon" href="apple-icon.png">
    <link rel="icon" sizes="192x192" href="android-icon.png">
</head>

Referensi

Untuk mengetahui lebih lanjut tentang cara membuat favicon, Anda bisa mengunjungi situs web seperti:

  • https://www.favicon.cc
  • https://www.realfavicongenerator.net/

Kesimpulan

Favicon adalah elemen penting untuk identitas visual situs Anda. Menggunakan favicon yang tepat dapat meningkatkan pengalaman pengguna dan memperkuat branding situs web Anda. Pastikan untuk menggunakan ukuran dan format yang sesuai agar favicon Anda tampil dengan baik di semua perangkat.

HTML Lists

HTML lists memungkinkan pengembang web untuk mengelompokkan sekumpulan item terkait dalam daftar.

Contoh Daftar HTML

HTML menyediakan tiga jenis daftar utama: daftar terurut (ordered list), daftar tidak terurut (unordered list), dan daftar deskriptif (description list). Berikut adalah penjelasan lebih lanjut mengenai masing-masing jenis daftar:

1. Daftar Tidak Terurut (Unordered List)

Daftar tidak terurut adalah daftar yang item-itemnya tidak diberi nomor. Daftar ini menggunakan tag <ul> untuk mendefinisikan keseluruhan daftar dan <li> untuk setiap item daftar.


<ul>
    <li>Item pertama</li>
    <li>Item kedua</li>
    <li>Item ketiga</li>
</ul>

Contoh Daftar Tidak Terurut:

  • Apel
  • Jeruk
  • Anggur

2. Daftar Terurut (Ordered List)

Daftar terurut adalah daftar yang item-itemnya diberi nomor atau urutan tertentu. Daftar ini menggunakan tag <ol> untuk mendefinisikan keseluruhan daftar dan <li> untuk setiap item daftar.


<ol>
    <li>Langkah pertama</li>
    <li>Langkah kedua</li>
    <li>Langkah ketiga</li>
</ol>

Contoh Daftar Terurut:

  1. Masukkan bahan-bahan
  2. Campurkan bahan-bahan
  3. Panggang dalam oven

3. Daftar Deskriptif (Description List)

Daftar deskriptif digunakan untuk menggambarkan item dengan penjelasan atau deskripsi lebih lanjut. Daftar ini menggunakan tag <dl> untuk mendefinisikan keseluruhan daftar, <dt> untuk item yang dijelaskan, dan <dd> untuk deskripsi atau penjelasan item tersebut.


<dl>
    <dt>HTML</dt>
    <dd>Bahasa markup untuk membuat halaman web</dd>
    <dt>CSS</dt>
    <dd>Bahasa untuk mendesain dan memformat halaman web</dd>
</dl>

Contoh Daftar Deskriptif:

HTML
Bahasa markup untuk membuat halaman web.
CSS
Bahasa untuk mendesain dan memformat halaman web.

4. Daftar Bersarang (Nested List)

Anda dapat membuat daftar bersarang dengan menempatkan daftar dalam daftar lainnya. Ini bisa diterapkan pada semua jenis daftar, baik terurut, tidak terurut, atau deskriptif.


<ul>
    <li>Item pertama</li>
    <li>Item kedua
        <ol>
            <li>Sub-item pertama</li>
            <li>Sub-item kedua</li>
        </ol>
    </li>
    <li>Item ketiga</li>
</ul>

Contoh Daftar Bersarang:

  • Item pertama
  • Item kedua
    1. Sub-item pertama
    2. Sub-item kedua
  • Item ketiga

Kesimpulan

HTML menyediakan berbagai jenis daftar yang memungkinkan pengelompokan data yang terkait dalam format yang mudah dibaca. Anda bisa memilih daftar yang sesuai dengan kebutuhan, apakah itu daftar yang diberi urutan, daftar tanpa urutan, atau daftar dengan deskripsi.

HTML Block and Inline Elements

Setiap elemen HTML memiliki nilai tampilan default, tergantung pada jenis elemen tersebut. Dua nilai tampilan yang paling umum adalah block dan inline.

Elemen Block

Elemen block adalah elemen yang secara otomatis memulai di baris baru dan mengambil seluruh lebar yang tersedia di dalam elemen induknya. Elemen block biasanya digunakan untuk konten yang ingin Anda tampilkan dalam format vertikal, satu per satu.

Contoh Elemen Block:

Beberapa elemen HTML yang secara default merupakan elemen block antara lain:

  • <div>
  • <p>
  • <h1> - <h6> (Tag Heading)
  • <section>
  • <article>
  • <header>
  • <footer>

Contoh Penggunaan Elemen Block:


<div>
    <h2>Judul Utama</h2>
    <p>Ini adalah paragraf pertama</p>
    <p>Ini adalah paragraf kedua</p>
</div>

Dalam contoh di atas, elemen <div>, <h2>, dan <p> semuanya merupakan elemen block. Setiap elemen tersebut dimulai di baris baru dan mengambil seluruh lebar yang tersedia.

Elemen Inline

Elemen inline adalah elemen yang tidak memulai di baris baru. Elemen inline hanya mengambil ruang sebesar kontennya dan tidak mempengaruhi tata letak elemen lainnya di sekitarnya.

Contoh Elemen Inline:

Beberapa elemen HTML yang secara default merupakan elemen inline antara lain:

  • <span>
  • <a>
  • <strong>
  • <em>
  • <img>

Contoh Penggunaan Elemen Inline:


<p>
    Ini adalah teks tebal dalam sebuah paragraf. 
    Ini adalah teks miring lainnya.
</p>

Dalam contoh di atas, elemen <strong> dan <em> adalah elemen inline. Mereka tidak membuat baris baru, melainkan hanya memberi efek pada bagian teks di dalam paragraf tersebut.

Perbedaan Antara Block dan Inline

Perbedaan utama antara elemen block dan inline adalah bagaimana mereka memengaruhi tata letak halaman:

  • Elemen Block: Memulai baris baru dan mengambil seluruh lebar kontainer induknya.
  • Elemen Inline: Tidak memulai baris baru, hanya mengambil ruang sebesar kontennya, dan memungkinkan elemen lainnya tampil berdampingan.

CSS untuk Mengubah Display

Anda juga bisa mengubah nilai display elemen dengan CSS. Nilai display dapat diatur menjadi block, inline, inline-block, flex, grid, dan lainnya. Berikut adalah contoh mengubah tampilan elemen dengan CSS:


/* Mengubah elemen menjadi block */
.element-block {
    display: block;
}

/* Mengubah elemen menjadi inline */
.element-inline {
    display: inline;
}

/* Mengubah elemen menjadi inline-block */
.element-inline-block {
    display: inline-block;
}

Contoh Penggunaan CSS:


<div class="element-block">Ini adalah div yang diubah menjadi block</div>
<span class="element-inline">Ini adalah span yang diubah menjadi inline</span>
<div class="element-inline-block">Ini adalah div yang diubah menjadi inline-block</div>

Kesimpulan

Elemen block dan inline memiliki perbedaan dalam cara mereka menampilkan konten di halaman. Elemen block digunakan untuk konten yang membutuhkan baris baru dan lebar penuh, sementara elemen inline digunakan untuk konten yang tidak memerlukan baris baru dan hanya memerlukan ruang sesuaikan dengan isi kontennya.

HTML class Attribute

Atribut class di HTML digunakan untuk memberikan kelas pada elemen. Kelas dapat digunakan untuk mengelompokkan elemen-elemen yang memiliki atribut atau karakteristik serupa, sehingga Anda bisa memberikan gaya atau melakukan manipulasi JavaScript pada elemen-elemen tersebut secara bersamaan.

Penggunaan Atribut class

Atribut class memungkinkan Anda untuk memberi nama pada sekumpulan elemen yang kemudian dapat dipilih dan diberi gaya (styling) melalui CSS atau diakses dengan JavaScript.

Contoh Penggunaan Atribut class:


<div class="header">
    <h1>Website Saya</h1>
</div>

<p class="intro">Ini adalah paragraf pengantar.</p>

Pada contoh di atas, elemen <div> diberi kelas "header" dan elemen <p> diberi kelas "intro". Kelas ini dapat digunakan untuk menambahkan styling menggunakan CSS atau untuk manipulasi elemen-elemen ini dengan JavaScript.

Menambahkan CSS Berdasarkan class

Dengan menggunakan kelas, Anda dapat memberikan gaya khusus pada elemen-elemen yang memiliki kelas tersebut. Berikut adalah cara menambahkan CSS untuk elemen dengan kelas tertentu.

/* Styling untuk elemen dengan class "header" */
.header {

Anda dapat memberikan lebih dari satu kelas pada elemen yang sama, dengan memisahkan nama-nama kelas menggunakan spasi. Ini memungkinkan elemen tersebut untuk memiliki gaya dari beberapa kelas sekaligus.

Contoh Beberapa Kelas dalam Satu Elemen:


<div class="header main">
    <h1>Website Saya</h1>
</div>

Pada contoh di atas, elemen <div> memiliki dua kelas, yaitu "header" dan "main". CSS yang diterapkan pada kedua kelas ini akan mempengaruhi elemen tersebut.

Contoh CSS dengan Beberapa Kelas:


/* Styling untuk elemen dengan class "header" */
.header {
    background-color: #4CAF50;
    color: white;
}

    font-size: 20px;
    text-align: center;
}

Jika sebuah elemen memiliki dua kelas seperti contoh di atas, maka kedua gaya dari kelas header dan main akan diterapkan pada elemen tersebut.

// Menambahkan kelas "active" ke elemen dengan ID "button"

Dengan menggunakan classList.remove(), Anda bisa menghapus kelas dari elemen yang sudah memiliki kelas tersebut, memungkinkan Anda untuk mengubah penampilan elemen secara dinamis.

Kesimpulan

Atribut class adalah alat yang sangat berguna dalam HTML untuk mengelompokkan elemen-elemen dan memberikan gaya atau manipulasi interaktif menggunakan CSS dan JavaScript. Dengan memberikan beberapa kelas pada satu elemen, Anda dapat menciptakan desain yang lebih fleksibel dan dinamis.

Atribut id dalam HTML

Atribut id di HTML digunakan untuk memberikan identifikasi unik pada elemen-elemen dalam halaman web. Setiap elemen yang diberi atribut id harus memiliki nilai yang unik dalam sebuah halaman, sehingga nilai id tidak boleh digunakan lebih dari sekali dalam halaman yang sama.

Penggunaan Atribut id

Atribut id sangat berguna ketika Anda perlu mengidentifikasi elemen secara unik, baik untuk menambahkan gaya CSS atau melakukan manipulasi menggunakan JavaScript. Anda bisa menggunakan id untuk memberi gaya pada elemen menggunakan CSS atau untuk mengakses elemen tersebut dalam JavaScript.

Contoh Penggunaan Atribut id:


<div id="header">
    <h1>Selamat Datang di Website Saya</h1>
</div>

<p id="intro">Ini adalah paragraf pengantar yang diperkenalkan dengan ID.</p>

Pada contoh di atas, elemen <div> memiliki atribut id="header" dan elemen <p> memiliki atribut id="intro". ID ini dapat digunakan untuk menambahkan gaya dengan CSS atau untuk memilih elemen-elemen ini di JavaScript.

Menambahkan CSS Berdasarkan id

Dengan menggunakan atribut id, Anda bisa memberi gaya khusus pada elemen tertentu. Dalam CSS, Anda bisa menargetkan elemen berdasarkan id menggunakan simbol hash (#) di depan nama ID.

Contoh CSS Berdasarkan id:


/* Styling untuk elemen dengan id "header" */
#header {
    background-color: #4CAF50;
    color: white;
    padding: 10px;
}

/* Styling untuk elemen dengan id "intro" */
#intro {
    font-size: 18px;
    color: #555;
}

Di sini, elemen dengan ID "header" akan memiliki latar belakang hijau dengan teks putih, dan elemen dengan ID "intro" akan memiliki ukuran font 18px dan warna teks abu-abu.

Menambahkan Atribut id Dinamis dengan JavaScript

Dengan JavaScript, Anda dapat menambahkan, mengubah, atau menghapus atribut id pada elemen-elemen HTML secara dinamis. Ini memungkinkan Anda untuk berinteraksi dengan elemen-elemen tersebut berdasarkan tindakan pengguna.

Contoh Mengubah id dengan JavaScript:


// Mengubah ID elemen dengan ID "intro" menjadi "intro-text"
document.getElementById("intro").id = "intro-text";

Dalam contoh ini, JavaScript digunakan untuk mengubah ID elemen dengan ID "intro" menjadi "intro-text" pada halaman.

Contoh Menambahkan ID dengan JavaScript:


// Menambahkan ID "new-element" pada elemen dengan ID "header"
document.getElementById("header").setAttribute("id", "new-element");

Dengan JavaScript, Anda dapat menambahkan atribut id baru ke elemen yang sebelumnya tidak memilikinya menggunakan setAttribute().

Contoh Menghapus ID dengan JavaScript:


// Menghapus ID dari elemen dengan ID "header"
document.getElementById("header").removeAttribute("id");

Dengan menggunakan removeAttribute(), Anda dapat menghapus atribut id dari elemen tertentu.

Memilih Elemen dengan id Menggunakan JavaScript

Atribut id sangat berguna saat Anda ingin memilih elemen tertentu dalam JavaScript. Anda dapat menggunakan metode getElementById() untuk memilih elemen berdasarkan id.

Contoh Mengakses Elemen dengan id Menggunakan JavaScript:


// Mengakses elemen dengan ID "header" dan menyimpannya dalam variabel
var headerElement = document.getElementById("header");

Metode getElementById() digunakan untuk mengakses elemen dengan ID "header" dan menyimpannya dalam variabel, yang kemudian bisa digunakan untuk manipulasi lebih lanjut.

Keunikan Atribut id

Atribut id harus bersifat unik dalam satu halaman. Artinya, tidak boleh ada dua elemen dengan id yang sama. Jika Anda memberikan dua elemen ID yang sama, hanya elemen pertama yang akan terpengaruh oleh gaya CSS atau JavaScript yang diterapkan pada ID tersebut.

Contoh yang Tidak Valid:


<div id="header">Header 1</div>
<div id="header">Header 2</div>

Di atas, dua elemen diberi id="header", yang tidak sesuai dengan aturan HTML. Hanya elemen pertama yang akan dipilih menggunakan ID tersebut. Ini bisa menyebabkan masalah dalam seleksi dan manipulasi elemen.

Kesimpulan

Atribut id sangat penting dalam HTML untuk memberikan identifikasi unik pada elemen-elemen. Ini memungkinkan Anda untuk memilih elemen-elemen tersebut dengan CSS dan JavaScript, serta berinteraksi dengan elemen-elemen di halaman secara dinamis. Pastikan bahwa setiap id yang Anda gunakan adalah unik di dalam halaman untuk menghindari masalah dalam manipulasi dan seleksi elemen.

HTML Iframes

Tag HTML <iframe> digunakan untuk menampilkan halaman web di dalam halaman web lainnya.

Sintaks HTML Iframe

Tag HTML <iframe> menentukan sebuah inline frame.

Inline frame digunakan untuk menyematkan dokumen lain di dalam dokumen HTML yang sedang aktif.

Sintaks untuk <iframe> adalah:


<iframe src="url" title="deskripsi"></iframe>

Tips: Sebaiknya selalu menyertakan atribut title untuk <iframe>. Atribut ini digunakan oleh pembaca layar (screen reader) untuk membaca konten dari iframe tersebut.

Iframe - Menentukan Tinggi dan Lebar

Gunakan atribut height dan width untuk menentukan ukuran iframe.

Tinggi dan lebar iframe biasanya ditentukan dalam satuan piksel:


<iframe src="demo_iframe.htm" height="200" width="300" title="Contoh Iframe"></iframe>

Atau, Anda bisa menambahkan atribut style dan menggunakan properti CSS height dan width:


<iframe src="demo_iframe.htm" style="height:200px;width:300px;" title="Contoh Iframe"></iframe>

Iframe - Menghapus Border

Secara default, iframe memiliki border di sekelilingnya.

Untuk menghapus border, tambahkan atribut style dan gunakan properti CSS border:


<iframe src="demo_iframe.htm" style="border:none;" title="Contoh Iframe"></iframe>

Dengan CSS, Anda juga bisa mengubah ukuran, gaya, dan warna border iframe:


<iframe src="demo_iframe.htm" style="border:2px solid red;" title="Contoh Iframe"></iframe>

Iframe - Target untuk Tautan

Iframe dapat digunakan sebagai target frame untuk sebuah tautan.

Atribut target dari tautan harus merujuk pada atribut name dari iframe:


<iframe src="demo_iframe.htm" name="iframe_a" title="Contoh Iframe"></iframe>
    
<p><a href="https://www.w3schools.com" target="iframe_a">W3Schools.com</a></p>

Ringkasan Bab

  • Tag HTML <iframe> digunakan untuk mendefinisikan sebuah inline frame.
  • Atribut src menentukan URL halaman yang akan disematkan.
  • Selalu sertakan atribut title (untuk pembaca layar).
  • Atribut height dan width menentukan ukuran iframe.
  • Gunakan border:none; untuk menghapus border di sekitar iframe.

HTML JavaScript

JavaScript membuat halaman HTML lebih dinamis dan interaktif. Dengan menggunakan JavaScript, Anda dapat mengubah konten, struktur, dan gaya halaman secara langsung tanpa memerlukan pemuatan ulang halaman. JavaScript memungkinkan Anda untuk menambahkan berbagai interaksi dengan pengguna, seperti respons terhadap klik tombol, pengisian formulir, atau pembaruan elemen halaman secara otomatis.

Menambahkan JavaScript ke dalam Halaman HTML

JavaScript dapat ditambahkan ke dalam halaman HTML dengan dua cara utama:

  • Inline JavaScript: Menambahkan JavaScript langsung dalam atribut HTML (misalnya, dalam event seperti onclick).
  • External JavaScript: Menambahkan JavaScript melalui file eksternal yang terpisah dari file HTML menggunakan tag <script>.

Contoh Inline JavaScript:


<button onclick="alert('Hello, World!')">Klik Saya</button>

Di contoh di atas, ketika tombol diklik, JavaScript akan menampilkan pesan "Hello, World!" di layar menggunakan metode alert().

Contoh External JavaScript:

Anda dapat menyimpan kode JavaScript dalam file terpisah dan menghubungkannya ke file HTML menggunakan tag <script>.


<script src="script.js"></script>

Dalam contoh ini, file JavaScript eksternal yang bernama script.js di-link ke halaman HTML Anda. Anda dapat menulis kode JavaScript di dalam file script.js, dan browser akan memuat dan menjalankannya saat halaman dimuat.

Menambahkan JavaScript di dalam Tag <script>

JavaScript juga dapat ditulis di dalam tag <script> dalam halaman HTML. Ini memungkinkan Anda untuk menambahkan JavaScript di bagian atas atau bawah halaman HTML.

Contoh JavaScript dalam Halaman HTML:


<script>
    function greetUser() {
        alert("Hello, welcome to my website!");
    }
</script>

<button onclick="greetUser()">Greet Me</button>

Di contoh di atas, fungsi greetUser() didefinisikan di dalam tag <script>, dan dipanggil ketika tombol "Greet Me" diklik.

Tempat Menempatkan Tag <script>

Tag <script> dapat ditempatkan di dua tempat utama dalam dokumen HTML:

  • Di dalam <head>: Untuk memuat JavaScript sebelum konten halaman dimuat.
  • Di dalam <body>: Untuk memuat JavaScript setelah konten halaman dimuat, yang seringkali lebih disarankan untuk menghindari pemblokiran pemuatan halaman.

Contoh Penempatan <script> di dalam <head>:


<head>
    <script>
        console.log('Script ini dimuat sebelum konten halaman');
    </script>
</head>

Contoh Penempatan <script> di dalam <body>:


<body>
    <h1>Selamat datang di halaman saya</h1>
    <script>
        console.log('Script ini dimuat setelah konten halaman');
    </script>
</body>

Event Handler dalam JavaScript

JavaScript memungkinkan Anda untuk menanggapi berbagai kejadian atau peristiwa (events) yang terjadi pada elemen HTML. Beberapa contoh peristiwa umum adalah klik, hover, perubahan teks, pengiriman formulir, dan banyak lagi.

Contoh Event Handler:


<button onclick="alert('Tombol diklik!')">Klik Saya</button>
<input type="text" oninput="console.log('Input teks!')">

Di sini, tombol akan menampilkan pesan ketika diklik, dan input teks akan mencetak pesan setiap kali pengguna mengetik sesuatu di dalam kotak input.

Menangani Peristiwa dengan JavaScript

Selain menambahkan atribut event langsung ke elemen, Anda juga dapat menangani peristiwa secara terpisah dengan menggunakan JavaScript.

Contoh Menangani Event dengan JavaScript:


<button id="myButton">Klik Saya</button>

<script>
    var button = document.getElementById('myButton');
    button.addEventListener('click', function() {
        alert('Tombol diklik dengan event listener!');
    });
</script>

Di sini, event click pada tombol diikat menggunakan addEventListener(), yang memungkinkan untuk menangani peristiwa secara lebih terstruktur.

JavaScript di dalam Formulir

JavaScript juga sering digunakan untuk menangani formulir. Anda bisa memvalidasi data sebelum dikirim, mencegah pengiriman formulir jika ada kesalahan, atau memperbarui konten halaman berdasarkan input formulir tanpa perlu memuat ulang halaman.

Contoh Validasi Formulir dengan JavaScript:


<form onsubmit="return validateForm()">
    <input type="text" id="name" required>
    <input type="submit" value="Kirim">
</form>

<script>
    function validateForm() {
        var name = document.getElementById("name").value;
        if (name == "") {
            alert("Nama tidak boleh kosong");
            return false;
        }
    }
</script>

Dalam contoh ini, JavaScript digunakan untuk memastikan bahwa formulir hanya dikirim jika kolom "Nama" tidak kosong.

Kesimpulan

JavaScript adalah bahasa pemrograman yang sangat penting dalam pengembangan web modern. Dengan JavaScript, Anda dapat membuat halaman web lebih interaktif dan dinamis. Anda dapat menambahkan interaksi dengan pengguna, memanipulasi konten halaman, serta menangani peristiwa dan formulir tanpa perlu memuat ulang halaman. Pastikan untuk memahami cara kerja JavaScript dan bagaimana menambahkannya ke halaman HTML Anda untuk meningkatkan pengalaman pengguna di situs Anda.

Elemen dan Teknik Layout HTML

Layout HTML digunakan untuk mengatur struktur halaman web. HTML menyediakan berbagai elemen dan teknik yang memungkinkan Anda untuk membuat layout yang fleksibel dan responsif. Elemen Layout Dasar

  • <aside> - Digunakan untuk mendefinisikan konten yang tidak langsung terkait dengan konten utama halaman.
  • <main> - Digunakan untuk mendefinisikan konten utama dari halaman yang relevan dengan topik utama.

Contoh Elemen Layout Dasar:


<header>

    <section>

CSS Grid Layout

CSS Grid Layout adalah salah satu teknik layout modern yang memungkinkan Anda untuk membuat layout responsif yang kompleks dengan mudah. Dengan grid, Anda dapat membagi halaman atau elemen menjadi kolom dan baris, dan menempatkan konten di dalamnya sesuai kebutuhan.

Contoh CSS Grid Layout:


<div class="grid-container">
    <div class="grid-item">Item 1</div>
    <div class="grid-item">Item 2</div>
    <div class="grid-item">Item 3</div>
    <div class="grid-item">Item 4</div>
</div>

<style>
    .grid-container {
        display: grid;
        grid-template-columns: auto auto auto;
        gap: 10px;
    }
    .grid-item {
        background-color: #f4f4f4;
        padding: 20px;
        border: 1px solid #ddd;
    }
</style>

Di atas, .grid-container mendefinisikan area grid yang memiliki tiga kolom, sementara .grid-item mendefinisikan setiap elemen di dalam grid tersebut.

Flexbox Layout

Flexbox adalah metode layout yang lebih sederhana daripada CSS Grid. Flexbox memungkinkan Anda untuk mendistribusikan ruang di antara elemen dalam sebuah kontainer dan mendukung pengaturan elemen secara dinamis. Ini sangat berguna untuk layout yang membutuhkan fleksibilitas dan responsivitas.

Contoh Flexbox Layout:


<div class="flex-container">
    <div class="flex-item">Item 1</div>
    <div class="flex-item">Item 2</div>
    <div class="flex-item">Item 3</div>
</div>

<style>
    .flex-container {
        display: flex;
        justify-content: space-between;
    }
    .flex-item {
        background-color: #f4f4f4;
        padding: 20px;
        border: 1px solid #ddd;
    }
</style>

Contoh ini menunjukkan bagaimana Flexbox memungkinkan elemen untuk terdistribusi secara merata di dalam kontainer, dengan justify-content: space-between yang memastikan ada ruang di antara elemen-elemen tersebut.

Responsive Layout dengan Media Queries

Untuk membuat layout yang responsif, Anda dapat menggunakan media queries di CSS. Media queries memungkinkan Anda untuk mengubah gaya berdasarkan lebar layar atau perangkat yang digunakan. Hal ini memungkinkan halaman web Anda untuk terlihat baik di berbagai ukuran layar, dari desktop hingga perangkat seluler.

Contoh Responsive Layout dengan Media Queries:


<style>
    .container {
        display: flex;
        flex-wrap: wrap;
    }
    .item {
        width: 100%;
        padding: 10px;
    }
    @media (min-width: 768px) {
        .item {
            width: 48%;
        }
    }
    @media (min-width: 1024px) {
        .item {
            width: 32%;
        }
    }
</style>
<div class="container">
    <div class="item">Item 1</div>
    <div class="item">Item 2</div>
    <div class="item">Item 3</div>
</div>

Di contoh di atas, @media digunakan untuk mengubah lebar elemen berdasarkan lebar layar. Pada layar kecil, elemen akan mengambil 100% lebar, pada layar lebih besar, elemen akan diatur untuk mengambil 48% dan 32% lebar dari layar, masing-masing.

Kesimpulan

Dalam HTML, Anda memiliki berbagai cara untuk membuat layout halaman web, baik dengan menggunakan elemen HTML standar seperti <header> dan <footer>, atau dengan menggunakan teknik CSS modern seperti Grid dan Flexbox. Memahami teknik layout ini memungkinkan Anda untuk membuat desain yang lebih fleksibel, responsif, dan mudah diubah sesuai dengan berbagai kebutuhan pengguna dan perangkat.

Desain Web Responsif

Desain web responsif adalah tentang membuat halaman web yang terlihat bagus di semua perangkat!

Desain web responsif akan secara otomatis menyesuaikan untuk berbagai ukuran layar dan viewport.

Apa itu Desain Web Responsif?

Desain Web Responsif adalah penggunaan HTML dan CSS untuk secara otomatis mengubah ukuran, menyembunyikan, memperkecil, atau memperbesar sebuah situs web, agar terlihat bagus di semua perangkat (desktop, tablet, dan ponsel):

Menyeting Viewport

Untuk membuat situs web responsif, tambahkan tag <meta> berikut ke semua halaman web Anda:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Tag ini akan mengatur viewport halaman Anda, yang memberi petunjuk kepada browser tentang bagaimana mengontrol dimensi dan skala halaman.

Gambar Responsif

Gambar responsif adalah gambar yang dapat menyesuaikan dengan ukuran browser yang berbeda.

Menggunakan Properti width

Jika properti CSS width diatur ke 100%, gambar akan responsif dan akan diskalakan naik atau turun:

<img src="img_girl.jpg" style="width:100%;">

Menggunakan Properti max-width

Jika properti max-width diatur ke 100%, gambar akan diskalakan ke bawah jika diperlukan, tetapi tidak akan diskalakan lebih besar dari ukuran aslinya:

<img src="img_girl.jpg" style="max-width:100%;height:auto;">

Menampilkan Gambar yang Berbeda Berdasarkan Lebar Browser

Elemen HTML <picture> memungkinkan Anda untuk mendefinisikan gambar yang berbeda untuk berbagai ukuran jendela browser.

Ubah ukuran jendela browser untuk melihat bagaimana gambar di bawah ini berubah sesuai dengan lebar layar:

<picture>
      <source srcset="img_smallflower.jpg" media="(max-width: 600px)">
      <source srcset="img_flowers.jpg" media="(max-width: 1500px)">
      <source srcset="flowers.jpg">
      <img src="img_smallflower.jpg" alt="Flowers">
    </picture>

Ukuran Teks Responsif

Ukuran teks dapat diatur dengan satuan "vw", yang berarti "viewport width". Dengan cara ini, ukuran teks akan mengikuti ukuran jendela browser:

<h1 style="font-size:10vw">Hello World</h1>

Media Queries

Selain mengubah ukuran teks dan gambar, sering juga digunakan media queries pada halaman web responsif.

Dengan media queries, Anda bisa mendefinisikan gaya yang sepenuhnya berbeda untuk berbagai ukuran browser.

Contoh:

Ubah ukuran jendela browser untuk melihat bahwa tiga elemen div di bawah ini akan ditampilkan secara horizontal di layar besar dan ditumpuk secara vertikal di layar kecil:


      .left, .right {
        float: left;
        width: 20%; /* Lebar adalah 20%, secara default */
      }

      .main {
        float: left;
        width: 60%; /* Lebar adalah 60%, secara default */
      }

      /* Gunakan media query untuk menambahkan breakpoint di 800px: */
      @media screen and (max-width: 800px) {
        .left, .main, .right {
          width: 100%; /* Lebar adalah 100%, ketika viewport 800px atau lebih kecil */
        }
      }

HTML Entities

Karakter-karakter yang sudah dipakai dalam HTML harus diganti dengan entitas:

  • < (kurang dari) = &lt;
  • > (lebih besar) = &gt;

Entitas Karakter HTML

Beberapa karakter bersifat reserved di HTML. Jika Anda menggunakan tanda kurang dari (<) atau lebih besar (>) dalam teks HTML Anda, browser mungkin akan mencampurinya dengan tag HTML.

Entitas nama atau nomor entitas bisa digunakan untuk menampilkan karakter-karakter yang reserved dalam HTML.

Entitas nama terlihat seperti ini:

&entity_name;

Nomor entitas terlihat seperti ini:

&#entity_number;

Untuk menampilkan tanda kurang dari (<), kita harus menulis: &lt; atau &#60;

Nama entitas lebih mudah diingat daripada nomor entitas.

Ruang Non-Pemutus

Salah satu entitas HTML yang sering digunakan adalah ruang non-pemutus: &nbsp;.

Ruang non-pemutus adalah ruang yang tidak akan terputus menjadi baris baru. Dua kata yang dipisahkan oleh ruang non-pemutus akan tetap berada di satu baris (tidak terputus ke baris berikutnya). Ini sangat berguna ketika memisahkan kata-kata bisa mengganggu.

Contoh:

  • Â§ 10
  • 10 km/h
  • 10 PM

Penggunaan umum lain dari ruang non-pemutus adalah untuk mencegah browser menghapus spasi di halaman HTML. Jika Anda menulis 10 spasi dalam teks, browser akan menghapus 9 di antaranya. Untuk menambahkan ruang yang sebenarnya dalam teks, Anda bisa menggunakan entitas karakter &nbsp;.

HTML Encoding (Set Karakter)

Untuk menampilkan halaman HTML dengan benar, browser web harus mengetahui set karakter yang digunakan.

Atribut charset HTML

Set karakter ditentukan dalam tag <meta>:

<meta charset="UTF-8">

Spesifikasi HTML5 mendorong pengembang web untuk menggunakan set karakter UTF-8.

UTF-8 mencakup hampir semua karakter dan simbol di dunia!

Unicode & Pertumbuhan Web

Referensi lengkap UTF-8.

Set Karakter ASCII

ASCII adalah standar pengkodean karakter pertama untuk web. Ini mendefinisikan 128 karakter berbeda yang dapat digunakan di internet:

  • Huruf-huruf Inggris (A-Z)
  • Angka (0-9)
  • Karakter khusus seperti ! $ + - ( ) @ < >.

Set Karakter ANSI

ANSI (Windows-1252) adalah set karakter asli Windows:

  • Identik dengan ASCII untuk 127 karakter pertama
  • Karakter khusus dari 128 hingga 159
  • Identik dengan UTF-8 dari 160 hingga 255
<meta charset="Windows-1252">

Set Karakter ISO-8859-1

ISO-8859-1 adalah set karakter default untuk HTML 4. Set karakter ini mendukung 256 kode karakter yang berbeda. HTML 4 juga mendukung UTF-8.

  • Identik dengan ASCII untuk 127 karakter pertama
  • Tidak menggunakan karakter dari 128 hingga 159
  • Identik dengan ANSI dan UTF-8 dari 160 hingga 255

Contoh HTML 4:

<meta http-equiv="Content-Type" content="text/html;charset=ISO-8859-1">

Contoh HTML 5:

<meta charset="ISO-8859-1">

Set Karakter UTF-8

  • Identik dengan ASCII untuk nilai dari 0 hingga 127
  • Tidak menggunakan karakter dari 128 hingga 159
  • Identik dengan ANSI dan 8859-1 dari 160 hingga 255
  • Lanjutan dari nilai 256 hingga 10.000 karakter
<meta charset="UTF-8">

HTML Uniform Resource Locators (URL)

Uniform Resource Locator (URL) adalah alamat yang digunakan untuk mengidentifikasi sumber daya (seperti halaman web, gambar, atau file) di internet. URL sangat penting dalam HTML karena mereka digunakan untuk menghubungkan halaman web satu dengan lainnya, serta mengarahkan browser ke lokasi tertentu di internet.

Struktur URL

URL memiliki struktur yang terdiri dari beberapa bagian, masing-masing memiliki fungsi tertentu. Struktur dasar URL adalah sebagai berikut:


    scheme://host:port/path?query#fragment

Bagian-bagian URL

  • Scheme: Menentukan protokol yang digunakan (misalnya http, https, ftp, dll). Contoh: https
  • Host: Nama domain atau alamat IP dari server tempat sumber daya berada. Contoh: www.example.com
  • Port: Nomor port yang digunakan untuk menghubungi server. Biasanya, ini tidak dituliskan jika menggunakan port default (misalnya 80 untuk HTTP dan 443 untuk HTTPS).
  • Path: Lokasi spesifik dari sumber daya di server. Contoh: /images/photo.jpg
  • Query: Parameter pencarian atau informasi tambahan yang dikirim ke server. Biasanya dimulai dengan tanda tanya ?. Contoh: ?id=123&name=John
  • Fragment: Bagian opsional yang merujuk pada bagian tertentu dari halaman web, dimulai dengan tanda pagar #. Contoh: #section2

Contoh URL

Berikut adalah beberapa contoh URL dengan berbagai elemen:


    https://www.example.com:443/images/photo.jpg?size=large#section1

Di sini:

  • Scheme: https
  • Host: www.example.com
  • Port: 443 (default untuk HTTPS)
  • Path: /images/photo.jpg
  • Query: ?size=large
  • Fragment: #section1

Menambahkan URL dalam HTML

Dalam HTML, URL digunakan untuk menghubungkan ke berbagai sumber daya, seperti halaman lain, gambar, file, atau situs eksternal. Berikut adalah beberapa cara menggunakan URL di HTML:

Menautkan ke Halaman Web

Anda dapat menggunakan elemen <a> untuk membuat tautan ke halaman web lain dengan menggunakan atribut href:


    <a href="https://www.example.com">Kunjungi Example</a>

Ini akan membuat tautan yang mengarah ke www.example.com.

Menampilkan Gambar Menggunakan URL

URL juga digunakan untuk menampilkan gambar dengan elemen <img>:


    <img src="https://www.example.com/images/photo.jpg" alt="Gambar Contoh">

Menautkan ke File dengan URL

URL juga digunakan untuk menautkan ke file seperti dokumen PDF, video, atau file lain:


    <a href="https://www.example.com/files/document.pdf">Unduh PDF</a>

URL Encoding

URL encoding adalah proses mengubah karakter-karakter khusus dalam URL menjadi format yang dapat diterima oleh browser. Beberapa karakter seperti spasi, tanda tanya, atau tanda & harus diencode karena mereka memiliki makna khusus dalam URL.

Contoh:


    https://www.example.com/search?q=hello world&lang=en

URL yang benar harus menjadi:


    https://www.example.com/search?q=hello%20world&lang=en

Di sini, spasi ( ) diubah menjadi %20 untuk memastikan bahwa URL dapat diproses dengan benar.

Penggunaan URL dalam HTML Forms

URL sering digunakan dalam formulir HTML untuk mengirimkan data ke server. URL digunakan dalam atribut action pada tag <form>:


    <form action="https://www.example.com/submit" method="post">
        <input type="text" name="name">
        <input type="submit" value="Kirim">
    </form>

Formulir ini akan mengirimkan data ke URL https://www.example.com/submit menggunakan metode POST.

HTML vs XHTML

HTML (HyperText Markup Language) dan XHTML (Extensible HyperText Markup Language) adalah dua versi standar yang digunakan untuk membuat halaman web. Meskipun keduanya memiliki tujuan yang sama, yaitu untuk mendesain dan menampilkan konten web, ada beberapa perbedaan penting antara keduanya.

Apa itu HTML?

HTML adalah bahasa markup standar yang digunakan untuk membuat halaman web. HTML menggunakan tag untuk mendefinisikan struktur dan elemen-elemen di halaman web, seperti teks, gambar, dan tabel. HTML lebih fleksibel dan lebih permisif dalam hal penulisan kode, yang membuatnya lebih mudah digunakan oleh pengembang baru.

Apa itu XHTML?

XHTML adalah versi yang lebih ketat dari HTML. XHTML diadopsi untuk membuat kode HTML lebih konsisten dan lebih terstruktur dengan mengikuti aturan XML (Extensible Markup Language). XHTML memerlukan penulisan tag yang lebih ketat dan mematuhi standar sintaks yang lebih ketat daripada HTML.

Perbedaan Utama Antara HTML dan XHTML

Aspek HTML XHTML
Tag Penutupan Beberapa tag tidak memerlukan penutupan, misalnya <br>, <img>. Semua tag harus ditutup, misalnya <br />, <img />.
Kasus Huruf Tag HTML tidak sensitif terhadap huruf besar/kecil. Tag XHTML harus ditulis dalam huruf kecil.
Aturan Sintaks HTML lebih fleksibel dalam hal aturan penulisan. XHTML mengikuti aturan sintaks XML yang lebih ketat dan harus valid.
Penggunaan Elemen yang Tidak Dikenal HTML lebih permisif dalam menangani elemen yang tidak dikenal. XHTML tidak akan menampilkan elemen yang tidak dikenal, karena harus valid menurut XML.
Penggunaan Doctype HTML dapat menggunakan berbagai tipe doctype, seperti <!DOCTYPE HTML>. XHTML menggunakan doctype yang lebih spesifik, seperti <!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">.

Contoh Kode HTML vs XHTML

Contoh Kode HTML:


<html>
    <head>
        <title>Contoh HTML</title>
    </head>
    <body>
        <h1>Selamat datang</h1>
        <p>Ini adalah contoh HTML</p>
    </body>
</html>

Contoh Kode XHTML:


<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns="http://www.w3.org/1999/xhtml">
    <head>
        <title>Contoh XHTML</title>
    </head>
    <body>
        <h1>Selamat datang</h1>
        <p>Ini adalah contoh XHTML</p>
    </body>
</html>

Perhatikan bahwa dalam XHTML, deklarasi XML di bagian atas halaman adalah wajib dan tag HTML harus ditulis dalam huruf kecil serta semua elemen harus ditutup dengan benar.

Kelebihan dan Kekurangan HTML dan XHTML

Kelebihan HTML:

  • Lebih fleksibel dalam penulisan kode.
  • Lebih mudah digunakan dan dipahami oleh pemula.
  • Didukung oleh semua browser tanpa masalah besar.

Kekurangan HTML:

  • Kurang terstruktur dan bisa berpotensi menyebabkan kesalahan dalam pengkodean yang sulit ditemukan.
  • Kurang ketat dalam hal validasi kode.

Kelebihan XHTML:

  • Lebih terstruktur dan valid menurut standar XML.
  • Memiliki sintaks yang lebih ketat, yang bisa mengurangi kemungkinan kesalahan dalam kode.

Kekurangan XHTML:

  • Lebih rumit untuk digunakan, terutama bagi pemula.
  • Tidak sepenuhnya didukung oleh semua browser, dan kesalahan sintaks dapat menyebabkan halaman tidak ditampilkan.

Kesimpulan

Baik HTML maupun XHTML memiliki kelebihan dan kekurangan masing-masing. HTML lebih mudah digunakan dan lebih fleksibel, sedangkan XHTML lebih ketat dan mematuhi standar XML yang lebih ketat. Pilihan antara HTML dan XHTML tergantung pada kebutuhan proyek Anda, namun saat ini HTML5 lebih sering digunakan untuk pengembangan web modern karena kemudahan dan dukungannya yang lebih baik terhadap berbagai teknologi web.

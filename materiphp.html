Apa itu PHP?

PHP adalah bahasa skrip server dan alat yang ampuh untuk membuat halaman Web yang dinamis dan interaktif. PHP merupakan alternatif yang banyak digunakan, gratis, dan efisien terhadap pesaing seperti ASP milik Microsoft.

Mengapa PHP?

  • PHP berjalan pada berbagai platform (Windows, Linux, Unix, Mac OS X, dll.)
  • PHP kompatibel dengan hampir semua server yang digunakan saat ini (Apache, IIS, dll.)
  • PHP mendukung berbagai macam database
  • PHP gratis. Unduh dari sumber resmi PHP: www.php.net
  • PHP mudah dipelajari dan berjalan secara efisien di sisi server

Apa yang baru di PHP 7

  • PHP 7 jauh lebih cepat dibandingkan rilis stabil populer sebelumnya (PHP 5.6)
  • PHP 7 telah meningkatkan Penanganan Kesalahan
  • PHP 7 mendukung Deklarasi Tipe yang lebih ketat untuk argumen fungsi
  • PHP 7 mendukung operator baru (seperti operator pesawat luar angkasa: <=>)

PHP Loops

Sering kali ketika Anda menulis kode, Anda ingin blok kode yang sama berjalan berulang-ulang dalam jumlah tertentu. Jadi, daripada menambahkan beberapa baris kode yang hampir sama dalam skrip, kita dapat menggunakan loop. Perulangan digunakan untuk mengeksekusi blok kode yang sama berulang kali, selama kondisi tertentu terpenuhi.

Dalam PHP, kita memiliki tipe loop berikut:

  • while- mengulang blok kode selama kondisi yang ditentukan benar
  • do...while- mengulang blok kode sekali, lalu mengulang loop selama kondisi yang ditentukan benar
  • for- mengulang blok kode sejumlah waktu tertentu
  • foreach- melakukan perulangan melalui blok kode untuk setiap elemen dalam array

Variabel PHP

Variabel adalah "wadah" untuk menyimpan informasi.

Dalam PHP, sebuah variabel dimulai dengan $tanda, diikuti dengan nama variabel:


            $x = 5;
            $y = "John";

Dalam contoh di atas, variabel $xakan menampung nilai 5, dan variabel $y akan menampung nilai "John".

Saat Anda menetapkan nilai teks ke suatu variabel, beri tanda kutip di sekitar nilai tersebut.

Tidak seperti bahasa pemrograman lain, PHP tidak memiliki perintah untuk mendeklarasikan variabel. Variabel dibuat saat Anda pertama kali menetapkan nilai padanya.

Anggaplah variabel sebagai wadah untuk menyimpan data.

Variabel PHP

Suatu variabel dapat memiliki nama pendek (seperti $xdan $y) atau nama yang lebih deskriptif ( $age, $carname, $total_volume).

Aturan untuk variabel PHP:

  • Variabel dimulai dengan $tanda, diikuti dengan nama variabel
  • Nama variabel harus dimulai dengan huruf atau karakter garis bawah
  • Nama variabel tidak dapat dimulai dengan angka
  • Nama variabel hanya dapat berisi karakter alfanumerik dan garis bawah (Az, 0-9, dan _)
  • Nama variabel peka huruf besar/kecil ( $agedan $AGEmerupakan dua variabel yang berbeda)

Variabel Keluaran

Contoh berikut akan menunjukkan cara mengeluarkan teks dan variabel:

Contoh berikut akan menghasilkan output yang sama seperti contoh di atas:


            Contoh

Contoh berikut akan menghasilkan jumlah dua variabel:

                public function message() {

                }
              }

PHP STRINGS

String adalah serangkaian karakter, seperti "Halo dunia!".

String dalam PHP diapit oleh tanda kutip ganda atau tanda kutip tunggal.


            echo "Hello";
            echo 'Hello';

Ada perbedaan besar antara tanda kutip ganda dan tanda kutip tunggal dalam PHP.

Tanda kutip ganda memproses karakter khusus, sedangkan tanda kutip tunggal tidak.

Tanda kutip ganda atau tunggal?

Anda dapat menggunakan tanda kutip ganda atau tunggal, tetapi Anda harus menyadari perbedaan antara keduanya.

String yang diapit tanda kutip ganda melakukan tindakan pada karakter khusus.

Misalnya ketika ada variabel dalam string, ia mengembalikan nilai variabel tersebut:


            Contoh
            Literal string yang dikutip ganda melakukan operasi untuk karakter khusus:
            $x = "John";
            echo "Hello $x";

String yang dikutip tunggal tidak melakukan tindakan tersebut, ia mengembalikan string seperti yang ditulis, dengan nama variabel:


            Contoh
            Literal string yang dikutip tunggal mengembalikan string sebagaimana adanya:
            $x = "John";
            echo 'Hello $x';

Fungsi PHP strlen()mengembalikan panjang suatu string.


            Contoh
            Mengembalikan panjang string "Halo dunia!":
            echo strlen("Hello world!");

Fungsi PHP str_word_count()menghitung jumlah kata dalam suatu string.


            Contoh
            Hitung jumlah kata dalam string "Halo dunia!":
            echo str_word_count("Hello world!");

Mencari Teks Dalam String

Fungsi PHP strpos()mencari teks tertentu dalam suatu string.

Jika kecocokan ditemukan, fungsi akan mengembalikan posisi karakter dari kecocokan pertama. Jika tidak ditemukan kecocokan, fungsi akan mengembalikan FALSE.


            Contoh
            Cari teks "dunia" dalam string "Halo dunia!":

Bilangan Bulat PHP

2, 256, -256, 10358, -179567 semuanya bilangan bulat.

Bilangan bulat adalah angka yang tidak memiliki bagian desimal.

Tipe data integer adalah angka non-desimal antara -2147483648 dan 2147483647 dalam sistem 32 bit, dan antara -9223372036854775808 dan 9223372036854775807 dalam sistem 64 bit. Nilai yang lebih besar (atau lebih kecil) dari ini akan disimpan sebagai float, karena melebihi batas integer.

Hal penting lainnya yang perlu diketahui adalah bahwa meskipun 4 * 2,5 adalah 10, hasilnya disimpan sebagai float, karena salah satu operan adalah float (2,5).

Berikut adalah beberapa aturan untuk bilangan bulat:

  • Suatu bilangan bulat harus memiliki setidaknya satu digit
  • Suatu bilangan bulat TIDAK boleh memiliki titik desimal
  • Suatu bilangan bulat bisa positif atau negatif
  • Bilangan bulat dapat ditentukan dalam tiga format: desimal (basis 10), heksadesimal (basis 16 - diawali dengan 0x), oktal (basis 8 - diawali dengan 0) atau biner (basis 2 - diawali dengan 0b).

PHP memiliki konstanta yang telah ditetapkan sebelumnya untuk bilangan bulat:

  • PHP_INT_MAX- Bilangan bulat terbesar yang didukung
  • PHP_INT_MIN- Bilangan bulat terkecil yang didukung
  • PHP_INT_SIZE- Ukuran integer dalam byte

PHP memiliki konstanta pradefinisi berikut untuk float (dari PHP 7.2):

  • PHP_FLOAT_DIG- Jumlah digit desimal yang dapat dibulatkan menjadi float dan kembali tanpa kehilangan presisi
  • PHP_FLOAT_EPSILON- Angka positif terkecil yang dapat direpresentasikan x, sehingga x + 1,0 != 1,0

PHP memiliki fungsi berikut untuk memeriksa apakah tipe suatu variabel adalah float:

  • is_float()
  • is_double()- alias dari is_float()

            Contoh

            var_dump(is_float($x));

PHP Casting

Terkadang Anda perlu mengubah variabel dari satu tipe data ke tipe data lain, dan terkadang Anda ingin variabel memiliki tipe data tertentu. Hal ini dapat dilakukan dengan casting.

Ubah Tipe Data

    Casting dalam PHP dilakukan dengan pernyataan berikut:
  • (string)- Mengonversi ke tipe data String
  • (int)- Mengonversi ke tipe data Integer
  • (float)- Mengonversi ke tipe data Float
  • (bool)- Mengonversi ke tipe data Boolean
  • (array)- Mengonversi ke tipe data Array
  • (object)- Mengonversi ke tipe data Objek
  • (unset)- Mengonversi ke tipe data NULL

untuk melakukan cast ke string, gunakan (string) pernyataan:


            $a = 5;       // Integer
            $b = 5.34;    // Float
            $c = "hello"; // String
            $d = true;    // Boolean
            $e = NULL;    // NULL

            $a = (string) $a;
            $b = (string) $b;
            $c = (string) $c;
            $d = (string) $d;
            $e = (string) $e;

            //To verify the type of any object in PHP, use the var_dump() function:
            var_dump($a);
            var_dump($b);
            var_dump($c);
            var_dump($d);
            var_dump($e);

Ubah ke integer

Untuk melakukan cast ke integer, gunakan (int) pernyataan:


            $a = 5;       // Integer
            $b = 5.34;    // Float
            $c = "25 kilometers"; // String
            $d = "kilometers 25"; // String
            $e = "hello"; // String
            $f = true;    // Boolean
            $g = NULL;    // NULL

            $a = (int) $a;
            $b = (int) $b;
            $c = (int) $c;
            $d = (int) $d;
            $e = (int) $e;
            $f = (int) $f;
            $g = (int) $g;

Untuk melakukan cast ke float, gunakan (float)pernyataan:


            $a = 5;       // Integer
            $b = 5.34;    // Float
            $c = "25 kilometers"; // String
            $d = "kilometers 25"; // String
            $e = "hello"; // String
            $f = true;    // Boolean
            $g = NULL;    // NULL

            $a = (float) $a;
            $b = (float) $b;
            $c = (float) $c;
            $d = (float) $d;
            $e = (float) $e;
            $f = (float) $f;
            $g = (float) $g;

Untuk melakukan cast ke boolean, gunakan (bool)pernyataan:


            $a = 5;       // Integer
            $b = 5.34;    // Float
            $c = 0;       // Integer
            $d = -1;      // Integer
            $e = 0.1;     // Float
            $f = "hello"; // String
            $g = "";      // String
            $h = true;    // Boolean
            $i = NULL;    // NULL

            $a = (bool) $a;
            $b = (bool) $b;
            $c = (bool) $c;
            $d = (bool) $d;
            $e = (bool) $e;
            $f = (bool) $f;
            $g = (bool) $g;
            $h = (bool) $h;
            $i = (bool) $i;

Jika suatu nilai adalah 0, NULL, salah, atau kosong, (bool) mengubahnya menjadi salah, jika tidak, benar.

Bahkan -1 diubah menjadi benar.

Untuk melakukan cast ke array, gunakan (array)pernyataan:


            $a = 5;       // Integer
            $b = 5.34;    // Float

PHP Constants

Konstanta seperti variabel, kecuali bahwa setelah didefinisikan, mereka tidak dapat diubah atau tidak dapat didefinisikan lagi.

Konstanta adalah pengenal (nama) untuk nilai sederhana. Nilai tersebut tidak dapat diubah selama skrip berlangsung.

Nama konstanta yang valid dimulai dengan huruf atau garis bawah (tanpa tanda $ sebelum nama konstanta).

Tidak seperti variabel, konstanta secara otomatis bersifat global di seluruh skrip.

Membuat Konstanta PHP

Untuk membuat konstanta, gunakan define() fungsi.

Sintaksis


            define(name, value);
    Parameternya:
  • Nama: Menentukan nama konstanta
  • Nilai: Menentukan nilai konstanta

            Buat konstanta dengan nama peka huruf besar/kecil :
            define("GREETING", "Welcome to W3Schools.com!");
            echo GREETING;

Kata Kunci PHP const

Anda juga dapat membuat konstanta dengan menggunakan constkata kunci.


            Contoh
            Buat konstanta peka huruf besar/kecil dengan constkata kunci:
            const MYCAR = "Volvo";
            echo MYCAR;

constmelawandefine()

  • consttidak dapat dibuat di dalam cakupan blok lain, seperti di dalam fungsi atau di dalam if pernyataan.
  • define dapat dibuat di dalam cakupan blok lain.

Array Konstan PHP

Dari PHP7, Anda dapat membuat konstanta Array menggunakan define()fungsi tersebut.


            Buat konstanta Array:

            define("cars", [
            "Alfa Romeo",
            "BMW",
            "Toyota"
            ]);
            echo cars[0];

Konstanta bersifat global

Konstanta secara otomatis bersifat global dan dapat digunakan di seluruh skrip.


            Contoh ini menggunakan konstanta di dalam suatu fungsi, 
            meskipun konstanta tersebut didefinisikan di luar fungsi tersebut:

            define("GREETING", "Welcome to W3Schools.com!");

            function myTest() {
            echo GREETING;
            }

            myTest();

Konstanta Ajaib PHP

PHP memiliki sembilan konstanta yang telah ditetapkan sebelumnya yang nilainya berubah tergantung di mana ia digunakan, dan karenanya ia disebut "konstanta ajaib". Konstanta ajaib ini ditulis dengan garis bawah ganda di awal dan akhir, kecuali untuk konstanta ClassName::class.

Konstanta Ajaib

  • __CLASS__ = If used inside a class, the class name is returned.
  • __DIR__ = The directory of the file.
  • __FILE__ = The file name including the full path.
  • __FUNCTION__ = If inside a function, the function name is returned.
  • __LINE__ = The current line number.
  • __METHOD__ = If used inside a function that belongs to a class, both class and function name is returned.
  • __NAMESPACE__ = If used inside a namespace, the name of the namespace is returned.
  • __TRAIT__ = If used inside a trait, the trait name is returned.
  • ClassName::class Returns the name of the specified class and the name of the namespace, if any.
Catatan:

Konstanta ajaib tidak peka huruf besar/kecil, artinya __LINE__akan mengembalikan nilai yang sama seperti __line__.
